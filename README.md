# Тестовое покрытие класса BooksCollector

Были реализованы следующие тесты для проверки функциональности класса:

### 1. Тесты инициализации объекта
- Проверка, что при создании объекта словари `books_genre` и `favorites` пусты
- Проверка, что список жанров инициализируется предопределенными значениями
- Проверка, что список жанров с возрастным ограничением 18+ заполнен корректно

### 2. Тесты добавления книг
- Проверка добавления книг с валидными названиями (параметризованный тест)
- Проверка, что книги с пустым названием не добавляются
- Проверка, что книги с названиями длиннее 40 символов не добавляются

### 3. Тесты работы с жанрами
- Проверка назначения корректных жанров книгам (параметризованный тест)
- Проверка, что нельзя назначить несуществующий жанр
- Проверка получения жанра по названию книги

### 4. Тесты фильтрации книг
- Проверка получения книг по конкретному жанру
- Проверка, что запрос несуществующего жанра возвращает пустой список
- Проверка получения полного списка всех книг
- Проверка, что для пустой коллекции возвращается пустой словарь

### 5. Тесты фильтра "для детей"
- Проверка, что в детскую подборку не попадают книги с "взрослыми" жанрами
- Проверка, что книги с жанрами 18+ исключаются из детского списка

### 6. Тесты работы с избранным
- Проверка добавления книг в избранное
- Проверка, что нельзя добавить несуществующую книгу в избранное
- Проверка удаления книг из избранного
- Проверка, что нельзя удалить несуществующую книгу из избранного
- Проверка получения списка избранных книг
- Проверка, что после удаления последней книги список избранного становится пустым
